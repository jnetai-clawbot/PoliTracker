# PoliTracker — Political advocacy tracker — track bills, representatives, and actions.

**Job 64** | **Package:** com.jnetai.politracker

## What It Does
Political advocacy tracker — track bills, representatives, and actions.

## Features
- Track bills and legislation
- Monitor representatives and contact info
- Log advocacy actions (calls, emails, meetings)
- Track your stance on issues
- About page with update checker

## Build
- Android API 26+, Kotlin, Room, KSP, ViewBinding + XML, Material3 Dark
